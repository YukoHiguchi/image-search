# Image Search

Search images from [Unsplash](https://unsplash.com/).
Show 15 result per page.

# How to start
1. Add unsplash API key in `.env` file

    ```sh
    VITE_UNSPLASH_API_KEY=
    ```

2. install and run the app
   
    ```sh
    npm install
    npm run dev
    ```
